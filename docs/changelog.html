v1.2.0 - May 14th, 2020

  • Added startup.html.
  • Added account setting pages.
  • Added Choices.js plugin for rich selects.
  • Added Quill plugin for rich text editor.
  • Added Dropzone plugin for drag & drop file uploads.

  • Improved documentation for icons.
  • Updated to Bootstrap v4.4.5.
  • Updated other dependencies.
  • Replaced navbar partial @@type attribute with @@container to support all container types.
  • Removed empty SCSS files.
  • Other small fixes and improvements.
