Buttons

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Bootstrap documentation

Soft

Landkit only

Creates a soft variant of a corresponding contextual button variation.

<button class="btn btn-primary-soft">Primary soft</button>
<button class="btn btn-secondary-soft">Secondary soft</button>
<button class="btn btn-success-soft">Success soft</button>
<button class="btn btn-danger-soft">Danger soft</button>
<button class="btn btn-warning-soft">Warning soft</button>
<button class="btn btn-info-soft">Info soft</button>

Pills

Landkit only

Creates a rounded variation of a button component.

<button class="btn btn-primary btn-pill btn-lg">
  Button pill
</button>

<button class="btn btn-success btn-pill">
  Button pill
</button>

<button class="btn btn-danger btn-pill btn-sm">
  Button pill
</button>

Circle

Landkit only

Turn any button into a circle with a .btn-rounded-cirle modifier.

<button class="btn btn-lg btn-rounded-circle btn-primary">
  <i class="fe fe-check"></i>
</button>

<button class="btn btn-rounded-circle btn-primary">
  <i class="fe fe-check"></i>
</button>

<button class="btn btn-sm btn-rounded-circle btn-primary">
  <i class="fe fe-check"></i>
</button>

<button class="btn btn-xs btn-rounded-circle btn-primary">
  <i class="fe fe-check"></i>
</button>
